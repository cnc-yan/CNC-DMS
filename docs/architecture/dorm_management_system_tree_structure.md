# 寮管理システム 要件定義 ツリー構造図

```text
寮管理システム
├─ 1. システム概要
│  ├─ 目的
│  │  ├─ 寮・部屋・入退寮履歴管理
│  │  ├─ 男性寮／女性寮管理
│  │  ├─ 日本社員・中国出張社員混在管理
│  │  ├─ 寮費自動計算
│  │  ├─ 長期利用警告・値上げ判断
│  │  ├─ 備品管理
│  │  └─ Excelデータ移行
│  └─ 将来本番利用前提
│
├─ 2. 前提条件
│  ├─ 約15戸
│  ├─ 主に3DK
│  ├─ 各部屋1名利用
│  ├─ 1戸最大3名
│  ├─ 男性寮／女性寮固定
│  └─ 所在地管理
│     ├─ 東京
│     └─ 大阪
│
├─ 3. 管理対象区分
│  └─ 入居者区分
│     ├─ 日本社員
│     │  ├─ 正社員
│     │  └─ 契約社員
│     └─ 中国出張社員
│        ├─ 短期出張
│        └─ 中長期出張
│
├─ 4. 寮・部屋管理
│  ├─ 4.1 寮管理
│  │  ├─ 管理項目
│  │  │  ├─ 寮ID
│  │  │  ├─ 寮名称
│  │  │  ├─ 住所
│  │  │  ├─ 間取り
│  │  │  ├─ 寮種別
│  │  │  └─ 備考
│  │  └─ 業務ルール
│  │     ├─ 男女混在禁止
│  │     ├─ 寮種別固定
│  │     └─ 原則変更不可
│  │
│  └─ 4.2 部屋管理
│     ├─ 部屋ID
│     ├─ 寮ID
│     ├─ 部屋名称
│     ├─ 部屋面積
│     └─ 定員
│
├─ 5. 入退寮管理
│  ├─ 5.1 入居履歴管理
│  │  ├─ 管理項目
│  │  │  ├─ 入居履歴ID
│  │  │  ├─ 社員ID
│  │  │  ├─ 寮ID
│  │  │  ├─ 部屋ID
│  │  │  ├─ 入寮日
│  │  │  ├─ 退寮日
│  │  │  ├─ 退寮理由
│  │  │  └─ 初回利用日
│  │  └─ 業務制約
│  │     ├─ 同一部屋重複入居禁止
│  │     ├─ 入居期間重複禁止
│  │     └─ 性別不一致禁止
│  │
│  └─ 5.2 初回利用日管理
│     ├─ 日本社員対象
│     ├─ 中国出張社員対象外
│     ├─ 通算利用期間算出
│     ├─ 長期利用警告
│     └─ 寮費値上げ判断
│
├─ 6. 寮費管理
│  ├─ 6.1 寮費算定
│  │  ├─ 部屋面積
│  │  ├─ 部屋種別
│  │  ├─ 日割り計算
│  │  └─ 将来拡張
│  │
│  └─ 6.2 寮費管理
│     ├─ 寮費ID
│     ├─ 社員ID
│     ├─ 対象年月
│     ├─ 算出金額
│     ├─ 算出根拠
│     └─ 確定ステータス
│
├─ 7. 備品管理
│  ├─ 7.1 備品マスタ
│  │  ├─ 備品ID
│  │  ├─ 備品名称
│  │  └─ 備品種別
│  │
│  ├─ 7.2 入居時備品準備
│  │  ├─ 入居者
│  │  ├─ 部屋
│  │  ├─ 備品
│  │  └─ 準備状況
│  │
│  ├─ 7.3 退去時備品処理
│  │  ├─ 廃棄
│  │  ├─ 保管
│  │  └─ 次回利用
│  │
│  └─ 7.4 備品保管管理
│     ├─ 保管場所
│     └─ ステータス
│
├─ 8. 空き室管理
│  ├─ 8.1 空き室判定
│  │  ├─ 入居履歴なし
│  │  ├─ 退寮済み
│  │  └─ 次回入居予定なし
│  │
│  └─ 8.2 機能要件
│     ├─ 空き室一覧表示
│     ├─ 空室／入居中表示
│     └─ 性別条件考慮
│
├─ 9. Excelデータ取り込み
│  ├─ 9.1 目的
│  │  └─ 初期データ移行
│  │
│  ├─ 9.2 対象データ
│  │  ├─ 寮マスタ
│  │  ├─ 部屋情報
│  │  ├─ 入居履歴
│  │  ├─ 現在入寮日
│  │  ├─ 初回利用日
│  │  └─ 寮費情報
│  │
│  └─ 9.3 機能要件
│     ├─ Excelアップロード
│     ├─ 項目マッピング
│     ├─ バリデーション
│     ├─ エラー一覧
│     └─ プレビュー
│
├─ 10. 非機能要件
│  ├─ 同時利用20名
│  ├─ 履歴5年以上保持
│  ├─ 論理削除
│  └─ 操作ログ
│     ├─ 入退寮
│     ├─ 寮費決定
│     └─ 備品処理
│
└─ 11. 生成AI開発評価観点
   ├─ 要件理解
   ├─ 設計力
   ├─ 実装力
   ├─ 保守性
   └─ 生産性
```
